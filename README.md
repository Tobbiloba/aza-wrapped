# aza-wrapped

Aza Wrapped - Your Spending Story. Upload your bank statement and get a personalized year-in-review experience like Spotify Wrapped.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Upload bank statements (CSV/XLSX)
- AI-powered insights and analysis
- Beautiful wrapped experience with 10 slides
- Share your wrapped on social media

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- Google Gemini AI
